Actividad 12: Etiqueta <a>

En esta actividad, exploraremos la etiqueta <a> y sus funciones para crear enlaces en páginas web.

Pasos de la actividad:

Crea un enlace a tu página web favorita y nombraló como "Mi página web favorita". Se abrirá en la misma pestaña que el documento HTML. Crea una nueva sección en el documento HTML debajo del bloque <br>. Crea un enlace interno a esa sección y nombraló como "Ir a la sección". Crea un enlace a otro documento HTML en la misma carpeta que este y que se abra en una nueva ventana o pestaña del navegador y nombraló como "Abrir en nueva ventana". Crea un enlace a una dirección de correo electrónico y nombraló como "Enviar correo electrónico". Crea una carpeta "img" en el directorio de este documento HTML y descarga una imagen en ella. Crea un enlace a esta imagen y nombraló como "Descargar Imagen".

Recuerda:

Utiliza el atributo href para especificar la dirección a la que se dirige cada enlace. Utiliza el atributo target para controlar cómo se abre el enlace. Utiliza el atributo id para indicar la identificación de la sección que luego será referida desde el href de la etiqueta <a>.
